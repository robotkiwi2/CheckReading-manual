# Check Reading이란?

**챌리딩(CheckReading)**은 6세~17세 영어 초심자를 위한 영어 읽기 능력 향상 솔루션입니다. 모바일 App 및 PC에서 사용할 수 있으며, AI와 시선추적 기술을 결합해 아이의 읽기 과정을 실시간으로 감독하고 능력을 키워줍니다.

## 주요 기능

* **고품질 TTS 읽기 자료 제공** — 다양한 수준과 길이의 영어 읽기 자료를 고품질 TTS(Text-to-Speech)로 제공합니다.
* **시선추적 기반 집중듣기 감독** — 시선추적 기술로 아이가 텍스트를 제대로 따라 읽고 있는지 감독합니다.
* **청크 기반 읽기 능력 향상** — 의미 단위(청크)로 나뉜 텍스트를 통해 읽기 이해력을 단계적으로 향상시킵니다.
* **실시간 읽기 감시** — 읽기 진행 상황을 실시간으로 모니터링하여 학습 데이터를 수집합니다.

## 주요 기술

* **AI 기반 글 생성 및 청크 구분** — AI가 학습 수준에 맞는 읽기 자료를 생성하고 청크 단위로 구분합니다.
* **AI 기반 음성 생성** — 자연스러운 영어 발음의 TTS 음성을 AI로 생성합니다.
* **모바일 시선추적 기술** — 별도 장비 없이 모바일 카메라로 시선을 추적합니다.
* **웹소켓 활용 읽기 데이터 전송** — 웹소켓을 통해 읽기 데이터를 실시간으로 서버에 전송합니다.

## 수익 모델

* **정기 구독 기반** — 월/연 단위 구독권을 통해 서비스를 이용합니다.
* **책 구매** — 개별 도서를 구매하여 이용하는 방식도 지원합니다.
