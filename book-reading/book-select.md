# 책 선택하기

읽을 책은 **책장** 또는 **서점**에서 선택할 수 있습니다.

---

## 책 선택 방법

1. 하단 탭에서 **책장** 또는 **서점**으로 이동합니다.
2. 원하는 책을 탭하면 오른쪽에 **책 상세 패널**이 열립니다.
3. 내용을 확인한 후 **\[책 읽기]** 버튼을 눌러 읽기를 시작합니다.

---

## 책 상세 패널 구성

### 상단 정보

- **표지 이미지**
- **Lv 배지**: 해당 책의 난이도 레벨
- **AR 지수**: 미국 Accelerated Reader 기준 독서 지수 (예: AR 4.2)
- **제목 / 저자 / 시리즈명**
- **분량**: 총 쪽수 (예: 86쪽)
- **등록일**: 서비스에 등록된 날짜

---

### 태그

책의 속성이 태그 형태로 표시됩니다.

| 태그 유형 | 예시 |
|-----------|------|
| 언어 | 영어, 한국어 |
| 종류 | 문학, 넌픽션 |
| 분량 | 단편, 중편, 장편 |
| 영어책 레벨 | 챕터북, 리더스북 등 |
| 퀴즈 재풀기 | 가능, 불가 |
| 장르 | #모험, #판타지, #역사 등 |

---

### TOC (목차)

- 책에 포함된 **챕터 목록**이 표시됩니다.
- 원하는 챕터 탭을 선택하면 해당 챕터부터 바로 읽기를 시작할 수 있습니다.

---

## 책 읽기 시작

**\[책 읽기]** 버튼을 누르면 시선 보정 화면으로 이동합니다.

{% hint style="info" %}
시선 보정 절차에 대한 자세한 내용은 [시선 보정하기](eye-calibration.md) 페이지를 참고하세요.
{% endhint %}
