# 서점

서점에서는 다양한 도서를 탐색하고, 자녀에게 도서를 추천할 수 있습니다.

---

## 선반(Shelf) 구성

서점 메인 화면은 여러 개의 선반으로 구성됩니다.

| 선반 유형 | 설명 |
|-----------|------|
| 자녀 프로필별 추천 선반 | 헤더에 원형 아바타와 함께 표시 (예: "홍길동이 좋아하는 SF") |
| 공통 선반 | 모든 사용자에게 동일하게 표시 (예: "이번 주 인기 도서") |

- 각 선반 제목 옆 **\[전체]** 버튼을 탭하면 해당 선반의 전체 도서를 확인할 수 있습니다.

---

## 책 상세 패널

책 카드를 탭하면 책 상세 패널이 열립니다.

- **표지 이미지** + **Lv 배지** + **AR 지수** (예: AR 4.2)
- 제목 / 저자 / 시리즈명
- 분량 (예: 86쪽) + 등록일 — 일반 텍스트로 표시
- **태그**: 영어 · 문학 · 장편 · 챕터북 · 퀴즈재풀기 + #장르태그 (한 줄로 통합 표시)
- **찜하기 하트 버튼**: 탭하면 현재 선택된 자녀 프로필의 색상으로 강조 표시

---

## 부모 전용: 권장도서 추가 기능

부모 계정으로 로그인한 경우, 책 상세 패널에서 자녀에게 도서를 직접 추천할 수 있습니다.

### 북마크 아이콘

- 책 상세 패널 상단의 **북마크 아이콘**을 탭합니다.
- 이미 추천한 책은 **노란 북마크**로 표시됩니다.

### 추가 절차

1. 북마크 아이콘 탭 → **자녀 프로필 선택 시트** 열림
2. 추천할 자녀 프로필 선택
3. **\[확인]** 버튼 탭
4. 이미 해당 자녀에게 추천된 책인 경우 팝업 표시:
   > "이미 홍길동에게 추천되어 있습니다. 진행하시겠습니까?"
   - **\[진행]**: 중복 추가
   - **\[취소]**: 취소

### 추가 결과

- 추가가 완료되면 자녀 책장의 **\[권장도서]** 탭에 즉시 반영됩니다.

{% hint style="info" %}
권장도서 추가 기능은 **부모 계정**에서만 사용할 수 있습니다.
{% endhint %}
