# 오늘의 현황

로그인 후 **프로필 목록** 화면에서 등록된 자녀들의 오늘 독서 현황을 한눈에 확인할 수 있습니다.

## 화면 구성

각 자녀 카드에는 다음 정보가 표시됩니다.

| 항목 | 설명 |
|------|------|
| 획득 점수 | 오늘 누적 독서 점수 |
| 읽은 시간 | 오늘 총 독서 시간 (분) |
| 읽은 단어 | 오늘 읽은 총 단어 수 |
| 시선 상태 바 | 읽기 / 스캐닝 / 이탈 비율 (녹색·노란색·빨간색) |

## 실시간 모니터링

자녀가 현재 뷰어를 열고 있으면 카드에 **초록 점(온라인)** 이 표시되고, **\[같이 보기]** 버튼이 나타납니다. 버튼을 탭하면 자녀가 읽고 있는 화면을 함께 볼 수 있습니다.

## 프로필 상세로 이동

자녀 카드를 탭하면 해당 자녀의 [프로필 상세](profile-detail.md) 화면으로 이동합니다.
