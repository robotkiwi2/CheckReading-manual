# 주간 리포트

매주 자녀의 독서 활동을 AI가 분석해 요약 리포트를 제공합니다.

## 리포트 확인하기

1. 프로필 상세 화면에서 **주간 리포트** 섹션으로 스크롤합니다.
2. 리포트 카드를 탭하면 상세 내용이 슬라이드업 패널로 열립니다.
3. 새 리포트는 **NEW** 배지로 표시됩니다.

## 리포트 구성

| 항목 | 내용 |
|------|------|
| 기간 요약 | 총점 · 전주 대비 변화 · 독서한 요일 |
| 시선 & 집중도 | 이번 주 평균 시선 비율, 전주 대비 변화 |
| 최고의 날 | 이번 주 집중도가 가장 높았던 요일 |
| 읽은 책 | 책별 진행률 · 점수 · WPM |
| AI 코멘트 | 이번 주 독서 패턴에 대한 분석 및 격려 메시지 |

## 소통에서 리포트 열기

채팅 화면에서 주간 리포트 알림 카드를 탭하면 해당 리포트 상세로 바로 이동합니다.
