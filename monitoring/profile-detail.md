# 프로필 상세

자녀의 독서 통계, 뱃지, 캘린더, 주간 리포트를 모아서 볼 수 있는 핵심 화면입니다.

## 화면 구성

프로필 카드 상단에는 자녀의 이름, 나이·학년·성별, 레벨, 등급(트로피), 또래 상위 % 가 표시됩니다.

하단 스크롤 영역은 다음 섹션으로 구성됩니다.

* [독서 통계](stats.md) — 누적/올해/이번달 점수 및 시선·집중도 차트
* [뱃지 & 등급](badges.md) — 획득 뱃지 목록과 등급 기준
* [독서 캘린더 & 일별 내역](calendar.md) — 월별 달력과 날짜별 읽기 기록
* [주간 리포트](weekly-report.md) — AI 코멘트 포함 주간 요약

## 즐겨찾기 (다른 사용자 프로필)

즐겨찾기한 다른 사용자의 자녀 프로필에서는 우측 상단에 하트 아이콘이 표시됩니다. 하트를 탭해 즐겨찾기를 추가하거나 해제할 수 있습니다.

## 설정 패널 (내 자녀 전용)

우측 상단 톱니 아이콘을 탭하면 자녀 프로필 설정 패널이 열립니다.

| 설정 항목 | 내용 |
|----------|------|
| 사진 교체 | 프로필 사진 변경 |
| 닉네임 | 자녀 닉네임 수정 |
| 비밀번호 변경 | 현재 비밀번호 확인 후 새 비밀번호로 변경 |
| 생년월·학년·성별 | 기본 정보 수정 |
| 선호 책 | 언어·종류·분량·영어레벨·퀴즈재풀기 태그 선택 |
| 책 추천 설정 | 추천 알림 및 방식 설정 |
| 책 뷰어 설정 | 읽기 탐지 세부 옵션 설정 |
| 알림 | 푸시 알림 온/오프 |
| 프로필 삭제 | 해당 자녀 프로필 영구 삭제 |

### 책 뷰어 설정 상세

| 항목 | 설명 |
|------|------|
| 페이지 이탈 감지 | 읽기 중 시선이 화면을 벗어날 때 감지 |
| 자동 시선 맞춤 | 시선 이탈 시 해당 문장으로 자동 복귀 유도 |
| 야간 읽기 제한 | 지정 시간 이후 읽기 종료 (밤 8시~12시 선택, 기본값: 10시) |
| 집중 듣기 도움 | 집중 듣기 모드에서 추가 도움 제공 |
| 읽기 속도 자동 조정 | 아이의 읽기 패턴에 맞춰 TTS 속도 자동 조정 |
| 권장도서 우선 | 권장도서를 책 목록 상단에 우선 표시 |

### 선호 책 설정 상세

태그를 탭해 선택/해제할 수 있습니다. 선택된 태그는 책 추천에 반영됩니다.

| 그룹 | 선택 항목 |
|------|----------|
| 언어 | 한국어 / 영어 |
| 종류 | 문학 / 넌픽션 / 기타 |
| 분량 | 단편 / 중편 / 장편 |
| 영어책 레벨 | 픽처북 / 스토리북 / 리더스북 / 얼리챕터북 / 미들챕터북 / 챕터북 / 노블 |
| 퀴즈 재풀기 | 가능 / 불가 |

{% hint style="warning" %}
프로필 삭제 시 해당 자녀의 모든 독서 기록이 영구적으로 삭제되며 복구할 수 없습니다.
{% endhint %}
