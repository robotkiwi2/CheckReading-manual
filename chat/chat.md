# 채팅

자녀별 채팅방에서 선생님·코치와 소통하고, 주간 리포트 알림을 받을 수 있습니다.

## 채팅방 목록

하단 탭바에서 **소통**을 탭하면 자녀별 채팅방 목록이 표시됩니다.

* 읽지 않은 메시지가 있으면 우측에 숫자 뱃지가 표시됩니다.
* 채팅방을 탭하면 대화 화면으로 이동합니다.

## 채팅 메뉴

채팅 화면 우측 상단 **⋮** 버튼을 탭하면 다음 옵션을 사용할 수 있습니다.

| 메뉴 | 설명 |
|------|------|
| 알림 끄기 | 해당 채팅방의 푸시 알림을 끕니다 |
| 채팅 지우기 | 대화 내용을 모두 삭제합니다 |

## 주간 리포트 알림

새 주간 리포트가 생성되면 채팅방에 알림 카드가 표시됩니다. 카드를 탭하면 [주간 리포트](../monitoring/weekly-report.md) 상세로 이동합니다.
