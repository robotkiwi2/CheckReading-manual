# 독서 수준 확인

자녀의 누적 독서 데이터를 바탕으로 산출된 독서 수준을 확인합니다.

## 레벨 확인 위치

* **프로필 카드** — 레벨 숫자와 등급 트로피로 한눈에 확인
* **프로필 상세 > 독서 통계** — 누적/올해/이번달 점수 비교

## 레벨 기준

독서 세션마다 쌓이는 독서 점수의 누적에 따라 레벨이 올라갑니다. 레벨은 아이의 읽기 속도, 시선 집중도, 퀴즈 정답률 등을 종합해 산출됩니다.

## 또래 비교

프로필 카드의 **상위 N%** 표시로 같은 연령대 사용자 중 자녀의 독서 수준을 확인할 수 있습니다.

## 등급 (트로피)

| 등급 | 설명 |
|------|------|
| 브론즈 | 독서 입문 단계 |
| 실버 | 꾸준한 독서 습관 형성 |
| 골드 | 높은 집중도와 이해력 |
| 플래티넘 | 상위 독자 수준 |
| 다이아몬드 | 최고 독서 수준 달성 |

트로피 아이콘을 탭하면 등급별 달성 기준 상세를 확인할 수 있습니다.
