# CheckReading 서비스 개요

> prototype·admin 모든 작업의 공통 참조 문서. 서비스 개념·용어·데이터 구조가 바뀌면 이 파일을 먼저 업데이트할 것.

---

## 한 줄 정의

**CheckReading**은 6~17세 영어 초심자를 위한 영어 읽기 솔루션으로,  
모바일 카메라 시선추적 기술로 아이가 책을 제대로 읽고 있는지 실시간 감독하고,  
부모가 함께 모니터링할 수 있는 서비스다.

---

## 사용자 유형

| 유형 | 설명 | 주요 화면 |
|------|------|----------|
| **부모 (parent)** | 자녀 계정을 관리하고 독서 상태를 모니터링 | 프로필, 서점, 책장, 내 정보 |
| **학생 (student)** | 책을 읽고 점수를 쌓는 실제 사용자 | 서점, 책장, 뷰어, 내 정보 |
| **관리자 (admin)** | 서비스를 운영·관리하는 내부 담당자 | admin 콘솔 전용 |

- 한 부모 계정에 자녀 프로필 여러 개 등록 가능
- 학생은 직접 로그인하거나 부모가 프로필을 선택해 진입
- `cr_user.type`으로 분기 (`'parent'` | `'student'`)

---

## 핵심 기능 개요

### 1. 책 읽기 (뷰어)

영어 원서를 뷰어에서 읽는다. 시선추적이 활성화되면 읽기 데이터가 실시간으로 수집된다.

**뷰어 진입 경로**
```
서점 책 상세 → [책 읽기]         → 1챕터부터
서점 책 상세 → TOC 챕터 클릭     → 해당 챕터부터
부모 프로필  → [같이 보기]       → 자녀 현재 위치부터 (모니터링 모드)
```

**뷰어 기능 토글 4종**

| 기능 | 설명 |
|------|------|
| 오디오 | TTS 음성 재생 (고품질 AI 음성) |
| 포인터 | 현재 읽는 위치를 시각적으로 표시 |
| 끊어읽기 | 의미 단위(청크)로 텍스트 분절 표시 |
| 시선 | 시선 추적 활성화 (캘리브레이션 필요) |

---

### 2. 시선 추적 (Eye Tracking)

별도 장비 없이 모바일 카메라로 시선을 추적하는 것이 핵심 기술.

**시선 상태 3종**

| 상태 | 색상 | 의미 |
|------|------|------|
| **Reading** | 🟢 emerald | 텍스트 라인을 순서대로 따라 읽는 중 |
| **Scanning** | 🟡 amber | 텍스트를 훑어보거나 빠르게 넘기는 중 |
| **Lost** | 🔴 red | 화면에서 시선이 벗어났거나 집중하지 않는 상태 |

세 상태의 비율(%)을 실시간으로 측정해 읽기 품질을 평가한다.  
**Lost 30% 초과** 시 경고 표시 (프로필 카드 테두리 amber, 부모에게 알림).

---

### 3. 캘리브레이션 (시선 보정)

**종류**

| 종류 | 점 수 | 소요시간 | 언제 |
|------|-------|----------|------|
| **1점 간이** | 1점 | ~3초 | 뷰어 열 때마다, 챕터 이동마다 |
| **5점 정밀** | 5점 | ~15초 | 정확도 저하 시 권고, 설정에서 수동 실행 |

**진입 플로우**: 뷰어 열기 → 얼굴 확인 → 1점 간이 캘리 → 읽기 시작  
`reader.html?skip_calib=1` 으로 캘리브레이션 건너뜀 (부모 모니터링 모드).

---

### 4. 알림 설정

| 그룹 | 항목 | 기본값 |
|------|------|--------|
| **읽기 이벤트** | 읽기 시작 알림 | ON |
| | 읽기 완료 알림 | ON |
| **중단 감지** | 읽기 중단 N분 이상 시 | ON, **5분** (3/5/10/15/20분) |
| **시선 상태** | Reading 비율 N% 이하로 저하 시 | ON, **50%** (30~70%) |
| | Lost 비율 N% 이상 급증 시 | ON, **30%** (20~50%) |
| **목표 & 습관** | 오늘 목표 달성 시 칭찬 알림 | ON |
| | 저녁 리마인더 (미읽음 시) | ON, **20:00** (18~22시) |
| | 주간 읽기 리포트 (일요일) | ON |
| **서비스** | 새 책 추천 알림 | OFF |
| | 레벨 업 알림 | ON |

---

### 5. 읽기 점수 (Reading Score)

읽기 세션마다 점수가 쌓인다. 시선 상태·읽기 속도·완독 여부 등을 종합 산출.

| 항목 | 설명 |
|------|------|
| 오늘 점수 | 당일 누적 |
| 이번 달 점수 | 월간 누적 |
| 올해 점수 | 연간 누적 |
| 총 점수 (totalScore) | 전체 누적 |

---

### 6. 레벨 & 등급

**레벨 (Lv.)**: 읽기 능력 수준. `Lv.1`(입문) ~ `Lv.6+`. 책마다 권장 레벨 표시.  
코드에서는 숫자 `1~6`으로 저장, UI에서 `'Lv.' + level` 형식으로 표시.

**영어책수준**: AR 레벨 형식 사용. 예: `"AR 4.2"` (책의 원어 난이도, 앱 레벨과 별개).

**등급 (Grade)**: 활동량·점수 기반 명예 등급.

| 등급 | 색상 |
|------|------|
| 브론즈 | orange |
| 실버 | gray |
| 골드 | yellow |
| 플래티넘 | cyan |
| 다이아몬드 | blue |

### 8. 도서 데이터 구조

```js
book = {
  title,          // 원작 제목
  author,         // 원작 저자
  creator,        // 내부 제작 담당자 (admin 전용)
  genre,          // 소설 / 판타지 / SF / 역사 / 자연 / 과학
  level,          // 숫자 1~6 (앱 레벨)
  engLevel,       // AR 레벨 문자열: "AR 4.2"
  type,           // 챕터북 / 소설 / 그림책 / 그레이디드 리더 / 단편집 / 학습서
  words,          // 단어 수 (분량)
  color,          // 표지 배경색 hex (단색) 또는 gradient
  publishStatus,  // 출판중 / 출판대기 / 출판중지 / 제작중
  modifiedDate,   // 수정일 YYYY-MM-DD (admin)
  reads,          // 총 읽기 횟수
  avgWpm,         // 평균 WPM
  avgScore,       // 평균 퀴즈 점수 (%)
  description,    // 책 소개
}
```

### 9. 프로필 데이터 구조

```js
profile = {
  id,             // 'hong' 등 고유 ID
  name,           // 닉네임
  initial,        // 아바타 표시용 첫 글자
  age,
  grade,          // '초3' 등 학년
  gender,         // '남' | '여'
  level,          // 'Lv.3' (문자열, 프로필은 문자열 형식 유지)
  gradeLabel,     // '골드' 등 명예 등급
  favorites,      // 찜한 책 수
  totalScore,     // 누적 읽기 점수
  color,          // 'linear-gradient(135deg,#667eea,#764ba2)'
  accent,         // '#667eea' (그라디언트 첫 색, 버튼 등에 사용)
  online,         // 현재 뷰어 열림 여부 (WebSocket)
  today: { score, time, words, eye: { reading, scanning, lost } },
  isOther,        // true: 즐겨찾는 다른 사용자의 자녀
}
```

---

### 10. 실시간 모니터링

WebSocket으로 자녀 뷰어 상태가 부모에게 전달됨 (프로토타입은 setInterval 시뮬레이션).  
Lost 30% 초과 → 카드 테두리 `ring-amber-400` + 부모 알림.

## localStorage 키 목록 (prototype 앱)

| 키 | 내용 |
|----|------|
| `cr_user` | 로그인 사용자 `{name, type, initial}` |
| `cr_profile` | 선택된 자녀 프로필 객체 전체 |
| `cr_reader_chapter` | 뷰어 진입 챕터 번호 (0-based) |
| `cr_openReport` | chat→profile-detail 주간리포트 자동 오픈 플래그 |
| `cr_admin` | admin 콘솔 로그인 정보 `{name, email}` |
